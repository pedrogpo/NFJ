# NFJuices

Purchase some juices :)

- [Figma Project](https://www.figma.com/file/ChphO2t7JUXKoVlLspFSTo/NFJ)

### Build

To build all apps and packages, run the following command:

```
cd nfj
yarn build
```

### Develop

To develop all apps and packages, run the following command:

```
cd nfj
yarn dev
```
